Basis Pengetahuan (Knowledge Base)

Teori & Rule Set Mocktail

Certainty Factor (CF) merepresentasikan tingkat keyakinan sistem terhadap fakta input pengguna atau probabilitas validitas sebuah aturan pakar.

Struktur Rule Set (3 Tahapan)

  • RS2 Status Keseimbangan Rasa (Kombinasi input Manis, Asam, Pahit)
  • RS3 Status Teknik Mixing (Kecocokan Teknik Mixing dengan Target Tekstur)
  • RS1 Rekomendasi Racikan (Hasil akhir menggabungkan Keseimbangan, Teknik, Dairy, dan Soda)

Rumus Perhitungan Logika

Nilai Keyakinan Fakta Gabungan (Evidence) dengan Operator AND:

CF(E) = min(CF Fakta1, CF Fakta2, ...)

Nilai Keyakinan Hipotesis dari sebuah Rule:

CF(H,E) = CF(E) × CF(Rule)

Penggabungan CF untuk Kesimpulan yang sama (MYCIN formula):

CFgabung = CFlama + CFbaru × (1 − CFlama)
